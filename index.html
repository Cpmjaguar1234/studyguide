10-5: Tangents

Exercises 1-2: Justify Tangent

Determine whether each segment is tangent to the given circle. Justify your answer.

1. $\overline{HI}$

Yes, $\overline{HI}$ is tangent.

Justification: Check using the converse of the Pythagorean Theorem.

$GI^2 + HI^2 \stackrel{?}{=} HG^2$

$9^2 + 40^2 \stackrel{?}{=} 41^2$

$81 + 1600 \stackrel{?}{=} 1681$

$1681 = 1681$. Since it's a right triangle, $\overline{HI} \perp \overline{GI}$.

2. $\overline{AB}$

No, $\overline{AB}$ is not tangent.

Justification: The radius is 4, so $AC = 4$. The hypotenuse $CB = 13$. The leg $AB = 12$.

$AC^2 + AB^2 \stackrel{?}{=} CB^2$

$4^2 + 12^2 \stackrel{?}{=} 13^2$

$16 + 144 \stackrel{?}{=} 169$

$160 \neq 169$. Since it's not a right triangle, $\overline{AB}$ is not perpendicular to the radius $\overline{AC}$.

10-8: Equations of Circles

Exercises 1-6: Write the Equation

Write the equation of each circle. The standard form is $(x - h)^2 + (y - k)^2 = r^2$.

1. Center at origin, radius 6

$(h, k) = (0, 0)$, $r = 6$

$(x - 0)^2 + (y - 0)^2 = 6^2$

$\mathbf{x^2 + y^2 = 36}$

2. Center at $(0, 0)$, radius 2

$(h, k) = (0, 0)$, $r = 2$

$(x - 0)^2 + (y - 0)^2 = 2^2$

$\mathbf{x^2 + y^2 = 4}$

3. Center at $(4, 3)$, radius 9

$(h, k) = (4, 3)$, $r = 9$

$(x - 4)^2 + (y - 3)^2 = 9^2$

$\mathbf{(x - 4)^2 + (y - 3)^2 = 81}$

4. Center at $(7, 1)$, diameter 24

$(h, k) = (7, 1)$, $d = 24 \implies r = 12$

$(x - 7)^2 + (y - 1)^2 = 12^2$

$\mathbf{(x - 7)^2 + (y - 1)^2 = 144}$

5. Center at $(-4, -1)$, passes through $(-2, 3)$

$(h, k) = (-4, -1)$. Need $r^2$.

Use distance formula for $r^2$: $(x - h)^2 + (y - k)^2 = r^2$

$(-2 - (-4))^2 + (3 - (-1))^2 = r^2$

$(-2 + 4)^2 + (3 + 1)^2 = r^2$

$2^2 + 4^2 = r^2 \implies 4 + 16 = r^2 \implies r^2 = 20$

Equation: $\mathbf{(x + 4)^2 + (y + 1)^2 = 20}$

6. Center at $(5, -2)$, passes through $(4, 0)$

$(h, k) = (5, -2)$. Need $r^2$.

Use distance formula for $r^2$: $(x - h)^2 + (y - k)^2 = r^2$

$(4 - 5)^2 + (0 - (-2))^2 = r^2$

$(-1)^2 + (2)^2 = r^2 \implies 1 + 4 = r^2 \implies r^2 = 5$

Equation: $\mathbf{(x - 5)^2 + (y + 2)^2 = 5}$
